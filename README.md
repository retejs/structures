**Rete.js extension**

## Key features

- **Mapping**: filter by nodes and connections
- **Sets**: union, difference, intersect
- **Subgraph**: children, parents, descendants, ancestors, orphans, siblings
- **Traverse**: roots, leaves, incomers, outgoers, successors, predecessors
